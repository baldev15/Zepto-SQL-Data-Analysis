# 🛒 Zepto SQL Data Analysis

## 📌 Project Overview

This project analyzes the **Zepto Grocery Products Dataset** using **PostgreSQL**. The dataset, sourced from **Kaggle**, was imported into PostgreSQL and analyzed using SQL to uncover valuable business insights related to pricing, discounts, inventory, and product categories.

The project demonstrates practical SQL skills through data exploration, data cleaning, and business-oriented analysis, making it an excellent portfolio project for aspiring Data Analysts.

---

## 🎯 Objectives

* Import and manage a real-world dataset in PostgreSQL.
* Perform data exploration and data cleaning.
* Analyze pricing, discounts, and inventory trends.
* Generate meaningful business insights using SQL.
* Practice real-world SQL querying and analytics.

## 📂 Dataset

**Source:** Kaggle

**Dataset:** Zepto Grocery Products Dataset

The dataset contains information about grocery products, including:

* Product Name
* Category
* MRP
* Discount Percentage
* Selling Price
* Quantity
* Weight
* Stock Availability

## 📊 Project Workflow

1. Created a PostgreSQL database and table.
2. Imported the Kaggle dataset into PostgreSQL.
3. Explored the dataset using SQL queries.
4. Cleaned the data by handling duplicates and missing values.
5. Performed business analysis using SQL.
6. Extracted actionable business insights.
