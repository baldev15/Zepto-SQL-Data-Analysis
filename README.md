🛒 Zepto SQL Data Analysis
📌 Project Overview

This project focuses on analyzing the Zepto Grocery Products Dataset using PostgreSQL. The dataset, sourced from Kaggle, was imported into PostgreSQL and analyzed using SQL to extract meaningful business insights related to pricing, discounts, inventory, and product categories.

The project demonstrates practical SQL skills, including data exploration, data cleaning, aggregation, filtering, and business-oriented analysis.

🎯 Objectives
Import a real-world retail dataset into PostgreSQL.
Perform data exploration and data cleaning.
Analyze product pricing and discount trends.
Evaluate inventory and stock availability.
Generate business insights using SQL queries.
